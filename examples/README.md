# examples/

Standalone sample sketches — one per folder — that demonstrate a complete
ESP32 project end-to-end. Most starter projects live directly inside
`ide.html` as in-memory templates (12 Python + 12 Arduino C++), but richer
multi-file examples belong here.

Structure per example:
- `README.md` — what it does, wiring diagram, how to run
- `main.py` or `sketch.ino` — the primary code
- `wiring.png` (optional) — a breadboard diagram

Empty for now. First candidates: a complete home-automation demo, a
voice-controlled robot, a multi-sensor classroom weather station.
