Connect Your ESP32

Plug your ESP32 into your computer with a USB cable. Then click "Select Port" below. Your browser will ask you to pick a serial port — choose the one that appears (it usually says "USB" or "CP210x" or "CH340" in the name).

First time? You may need a USB driver. Most ESP32 boards use the CP2102 or CH340 chip. Search for "[your chip name] driver download" if the board doesn't show up.

Send Your Code to the Board

This sends your finished code to the ESP32 so it starts running on the board. Works in Chrome and Edge. If the board messages panel is busy, it'll pause while we send.

First time? You need a compiled .bin file (a packaged version of your code). We're working on making this happen automatically — for now, drop in a .bin you already have below.

Advanced options (you probably don't need these)

The default 0x10000 works for almost every ESP32 project. Change only if you know why.
